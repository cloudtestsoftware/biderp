Know about creating project from Quote

Once you completed your Quote as cost basis, you might have identified all parts and resources needed for each Quote jobs. Now if this initial estimation is acceptable, then you can convert this Quote into a project by clicking "Convert Quote To Project" button. This will create a project record copying all master jobs and quote jobs to the project along with parts and resource list.

Make sure that you convert Quote to project when you are finally ready with your Quotation and no more changes are needed.

Step -2: Open the Project and review

  • Select "Project Setup" app from Apps menu
  • Search for the project with the name same as Quote
  • Look at Master Jobs and Job List using filter as "Project Title" as Quote name
  • Under each job, look at their parts and resources using tabs

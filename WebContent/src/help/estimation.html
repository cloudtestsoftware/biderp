Know about seed data setup

Below are steps to start a new project

Choose "Seed Data" menu item in the App menu list

Step -2: Create project template

  • Click on ProjectCode and then click search button without any filter in the input box.
  • Add a project code record
  • Add main job code associated with project code
  • Add Sub Job code for each main Job code

If you are working with large construction project then each of the Project Code will be one main job like " Turbine Installation Project" , "Generator Installation Project" etc.
